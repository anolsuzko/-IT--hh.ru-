## О проекте
Цель исследования — проанализировать рынок IT-вакансий России на hh.ru за период
май 2025 – июнь 2026: выявить закономерности в востребованности специальностей и
навыков, уровнях оплаты труда и сезонной динамике, а также построить модель
прогнозирования зарплаты на основе характеристик вакансии.

Рассматривались 7 специализаций: ML-инженер, Data Scientist, Frontend, Backend,
Fullstack, системный аналитик, DevOps.

Были сформулированы и проверены четыре гипотезы:
1. | H1 | Опыт работы — ключевой фактор влияния на зарплату | Подтверждена |
2. | H2 | SQL — самый универсальный и востребованный навык на рынке IT | Подтверждена |
3. | H3 | Количество вакансий имеет выраженную сезонную составляющую в течение года | Подтверждена частично |
4. | H4 | Data Scientist демонстрирует наиболее высокий темп роста спроса среди анализируемых специальностей | Подтверждена |

## Методология
# Сбор данных. Двухэтапный парсинг hh.ru: сначала постранично собирались карточки
вакансий из поиска (паджинация, до 20 страниц на специальность, по 100 вакансий на
странице), затем по идентификатору каждой вакансии выполнялся дополнительный запрос
к странице вакансии для получения требуемых навыков и точной даты публикации.
Собрано 601 запись по 10 признакам: тип вакансии, название, мин./макс. зарплата,
тип зарплаты, требуемый опыт, город, компания, навыки, дата публикации.

# Очистка и предобработка.
- Пропуски в зарплате (76% по минимуму, 82,5% по максимуму) не заполнялись —
  работодатели намеренно не указывают вилку. Создан бинарный признак "зарплата
  указана" (1/0), чтобы не потерять эту информацию при дальнейшем анализе.
- Остальные числовые пропуски заменены на −1, строковые — на "Не указаны".
- Удалено 3 пары полных дубликатов строк (было 601 стало 598).
- Аномальные значения зарплаты (артефакты парсинга порядка 10^299) заменены на
  пропуски методом межквартильного размаха (IQR); значение 450 000 ₽ оставлено как
  реальная зарплата Senior-специалиста.
- Устранены несогласованности в названиях городов ("город Новосибирск" -
  "Новосибирск", "городской округ Химки" - "Химки", "Киров (Кировская область)" -
  "Киров").
- Названия навыков нормализованы: 646 уникальных значений сведены к 243
  (объединены синонимы вида Go/Golang, React/ReactJS, JavaScript/JS).

# Анализ. EDA с оценкой распределения (skewness/kurtosis), частотный и
комбинаторный анализ навыков, анализ временной динамики (тренд, сезонная
декомпозиция с периодом 3 месяца), корреляционный анализ (Пирсон), сравнение
по грейдам и городам, линейная регрессия ('scikit-learn', признаки
стандартизированы) для прогнозирования зарплаты.

## Основные результаты

# Навыки. Безусловный лидер — SQL: 172 упоминания (7,2% всех упоминаний),
первое место независимо от специальности. В топ-5 также API-технологии, Git,
Python, Docker. Топ-15 навыков покрывают 45,2% всех упоминаний. Самая частая
пара навыков — API-технологии + SQL (65 совместных упоминаний); Docker и Linux
также часто встречаются вместе (57% упоминаний Docker). Наборы навыков чётко
отражают специфику роли: Python универсален для Backend/DS/ML/DevOps, у Frontend
лидируют JavaScript/React/TypeScript, у DevOps — Linux/Kubernetes/Docker.

# Зарплаты. Медиана минимальной зарплаты — 179 500 RUB, средняя максимальной —
238 286 RUB; после финальной очистки средняя зарплата — 191 425 RUB, медиана —
200 000 RUB, основная масса вакансий — в диапазоне 150–250 тыс. RUB. По грейдам:
Junior - 89 750 RUB ; Middle - 201 140 RUB ; Senior - 310 812 RUB ; Lead - 303 000 RUB
(корреляция грейда с зарплатой r = 0,264). По городам Москва занимает только
6-е место (212 тыс. RUB) — выше неё Екатеринбург (260 тыс. RUB), а также
Новосибирск, Ижевск и Уфа, что указывает на конкурентные ставки в регионах.

# Динамика рынка. Пик публикаций — сентябрь (117 вакансий), резкий провал —
октябрь (1 вакансия); общий линейный тренд слабо отрицательный (−0,4 вак./мес.)
при выраженных сезонных всплесках. Сезонная декомпозиция с периодом 3 месяца
показала, что трёхмесячный цикл не является доминирующим паттерном рынка.
По темпу роста лидирует Data Scientist (+1,3 вак./мес.), на втором месте
Frontend (+0,75 вак./мес.); единственная специализация со снижением спроса —
Fullstack (−0,05 вак./мес.).

# Регрессионная модель. R^2 = 0,24 на тестовой выборке, MAE = 50 788 RUB. Наибольший
положительный вклад в зарплату вносят опыт работы и город; вакансии без
указания конкретных навыков в среднем платят больше (208 тыс. RUB против 185 тыс. RUB
с подробными требованиями) — вероятно, такие позиции реже детализируют условия.
Зафиксирован также слабый эффект дня публикации (пик — пятница, минимум — среда).

## Ограничения исследования

- Зарплата раскрыта только в 23,8% вакансий — регрессионная модель и часть
  статистики по зарплате построены на этой подвыборке и не описывают рынок
  труда в целом.
- Признак "требуемый опыт" в собранных данных принимает фактически только два
  значения (не указано / 3 года) — это особенность парсинга, а не реального
  распределения опыта на рынке, и влияет на интерпретацию связи опыта с зарплатой.
- Анализ сезонности построен на данных за один год — недостаточно для надёжных
  выводов о многолетней цикличности рынка.
